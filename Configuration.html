Archlinux安装后配置

创建交换文件

在分区时没有创建swap区时执行。 

 #fallocate -l 512M /swapfile		分配块空间用于交换文件
 #chmod 600 /swapfile				更改权限
 #mkswap /swapfile					设置交换文件
 #swapon /swapfile					启用
 #nano /etc/fstab
  /swapfile none swap defaults 0 0	最后一行写入

新建用户并配置sudo

 #useradd -m -g users -s /bin/bash 用户名
 #passwd 用户名
 #nano /etc/sudoers
  用户名 ALL = (ALL)ALL 			最后一行写入

图形界面的安装

  1. 显卡驱动 xf86-video-intel
  2. 图形服务 xorg xorg-xinit
  3. 触板驱动 xf86-input-synaptics

包源

用root打开 /etc/pacman.conf ，在末尾写入 

 [archlinuxcn]
 SigLevel = Never
 Serrer = https://mirrorstuna.tsinghua.edu.cn/archlinuxcn/$arch

生效 #sudo pacman -S archlinuxcn-keyring 同步 #sudo pacman -Sy

yaourt

安装 #sudo pacman -S yaourt 更新 #yaourt -Syu --aur

中文支持

中文输入法安装 fcitx fcitx-im fcitx-configtool fcitx-googlepinyin 配置（在.xinitrc文件中加入） 

export LANG=zh_CN.UTF-8
export LANGUAGE=zh_CN:en_US
export LC_CTYPE=en_US.UTF-8
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"

中文字体 wqy-microhei 文泉译微米黑，无衬线字体

终端 oh-my-zsh

准备并设为默认 

 #pacman -S zsh git wget
 #chsh -s $(which zsh)

安装 oh-my-zsh 

 #sh -c "$(wget https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

配置文件（.zshrc）

  1. 主题 ZSH_THEME="rkj-repos
  2. 插件 plugins=(git)
  3. 简化命令 alias ll = 'ls -l'

更新

  1. 手动 #$upgrade_oh_my_zsh 或 #source .zshrc
  2. 自动 在.zshrc中添加 DISABLE_UPDATE_PROMPT = true
  3. 卸载 #uninstall_oh_my_zsh
